# 大连理工大学 - 科研启动计划 (DUT AI Starter Plan)

欢迎来到大连理工大学科研启动计划项目！

## 📖 项目简介

本项目旨在为大连理工大学对人工智能（AI）充满热情的学生，提供一个全面、系统化的学习与实践平台。我们精心整理了从入门基础到领域前沿的各类学习资源，希望能帮助您：

-   **快速入门**：轻松掌握AI的基础知识和必备工具。
-   **深入探索**：系统学习机器学习、深度学习等核心领域。
-   **提升能力**：结合实践项目，锻炼科研和工程能力。

无论您是编程新手，还是希望在AI领域深耕的同学，这里都有适合您的内容。

## 🚀 我们的目标

-   **资源整合**：汇集高质量的课程视频、电子书、开源项目和学术论文。
-   **路径规划**：提供清晰的学习路线图，让您的学习过程不再迷茫。
-   **社区互助**：建立一个积极的学习社区，共同解决难题，分享经验。

## 📚 主要内容

本仓库网站包含了以下几个核心模块：

1.  **新手入门指南**
    -   为初学者提供环境配置、基础工具使用等入门指导。

2.  **课程资源总览**
    -   **基础课程**：涵盖数学基础、Python编程、数据结构等。
    -   **核心技术**：机器学习、深度学习、计算机视觉、自然语言处理等。
    -   **名校公开课**：精选吴恩达、李宏毅等知名教授的经典课程。

3.  **项目实践**
    -   提供从简单到复杂的实践任务，助您学以致用。

4.  **学术前沿**
    -   分享最新的学术会议、经典论文和前沿动态。
